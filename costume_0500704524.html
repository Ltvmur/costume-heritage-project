Language Model Results

I used ChatGPT, Gemini, and LlaMa 2 Family to analyze this costume. All three models agreed that the costume was designed for the play Isabella, tre caravelle e un cacciaballe and reflected a stylized version of late 15th to early 16th-century Spanish court fashion.

SPARQL Query

SELECT DISTINCT ?item ?label ?material ?subject
WHERE {
  {
    ?item rdfs:label ?label .
    FILTER(REGEX(LCASE(STR(?label)), "costume"))
  }
  UNION
  {
    ?item a-dd:hasType ?type .
    FILTER(REGEX(STR(?type), "costume", "i"))
  }
  OPTIONAL { ?item a-dd:hasMaterial ?material . }
  OPTIONAL { ?item a-cd:hasSubject ?subject . }
}
ORDER BY ASC(?label)
LIMIT 10

Conclusion

This project demonstrates how knowledge graphs and language models can work together to enrich cultural heritage data. The costume is not only a visual artifact, but also a narrative element linked to Italian theater and political satire.
